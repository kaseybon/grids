Thoughts

One thing I like about this grid system is you can pick the number of columns you are going to need and generate the CSS and sample HTML. You are also able to have columns that span the width of multiple columns. The CSS is easy to edit and the grids are also able to be nested.

But... I think this grid system was meant to be for layouts only. Each row has to be wrapped in a div or else you get some funky margins. This makes it really difficult to use for the product grid above. It also has only one media query so it's either the columns span the full width or they are set at whatever column count you chose, so in my example: 4. That's a big jump and at certain screen sizes before the cards go full screen they look squished.
